# 📱 영숙 — 회사 목표, 각 에이전트의 개인 목표, 최근 의사결정, 메모리를 검토하여 현재 가장 가치 있는 단일 작업을 결정하고, 해당 작업을 실행할 에이전트를 1~2명에게 분배하여 실행 계획을 수립하라.

📱 **가장 중요한 작업 (Action Plan)**

현재 시스템은 **'가장 중요한 작업'**을 수행하기 위해 **'회사 시스템'**의 맥락을 이해하고, **'회사 시스템'**에 저장된 정보(특히 '회사 시스템'의 파일 구조나 기존 작업 목록)를 참조해야 합니다.

**현재 제공된 컨텍스트만으로는 구체적인 '가장 중요한 작업'이 무엇인지 알 수 없습니다.**

따라서, 귀하가 **어떤 종류의 작업을** 수행하기를 원하는지 명확하게 지시해 주셔야 제가 그에 맞는 계획을 세우고 실행할 수 있습니다.

**어떤 작업을 원하시나요? (예: 다음 주 마케팅 계획 세우기, 보고서 초안 작성하기, 특정 코드 디버깅하기 등)**

구체적인 목표를 알려주시면, 제공된 모든 컨텍스트와 지침을 활용하여 최적의 실행 계획을 수립하겠습니다.
