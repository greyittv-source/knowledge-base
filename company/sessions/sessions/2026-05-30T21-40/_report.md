# 📝 CEO 종합 보고서

📱 **분석 결과 및 실행 계획**

현재 시스템은 사용자로부터 명시적인 작업 요청을 받지 않았으므로, 시스템이 스스로 다음 단계를 제안하고 계획을 수립하는 방식으로 작동합니다.

**1. 현재 상황 분석:**
*   **요청:** 명시적인 작업 요청 없음. (시스템 초기 상태)
*   **목표:** 사용자에게 가장 유용하고 생산적인 다음 단계를 제안하여 작업을 시작하도록 유도해야 함.
*   **가용 정보:** 시스템 내부에 저장된 과거의 대화 기록 및 설정 정보(분석 결과)만 존재함.

**2. 제안할 다음 단계 (Action Plan):**
사용자가 현재 어떤 종류의 작업을 원하시는지 파악하기 위해, 시스템이 제공할 수 있는 일반적인 기능 목록을 제시하고 선택을 유도해야 합니다.

**3. 실행 계획 (Execution Plan):**

1.  **상황 확인 요청:** 사용자에게 현재 시스템이 어떤 종류의 작업을 수행하기를 원하는지 질문합니다.
2.  **기능 제시:** 시스템이 제공할 수 있는 주요 기능 목록(예: 콘텐츠 생성, 데이터 분석, 계획 수립, 정보 요약 등)을 제시합니다.
3.  **작업 시작:** 사용자가 원하는 작업을 선택하면, 해당 작업에 맞춰 구체적인 계획을 수립하고 실행을 시작합니다.

**4. 실행 (Execution):**

현재로서는 사용자 입력이 없으므로, 다음 단계는 사용자에게 질문하는 것입니다.

---

**다음 단계:**

안녕하세요. 현재 시스템은 준비가 완료되었으며, 어떤 종류의 작업을 도와드릴까요?

예를 들어, 다음과 같은 작업을 요청하실 수 있습니다:

1.  **정보 요약:** 특정 주제에 대한 정보를 요약해 주세요.
2.  **콘텐츠 생성:** 특정 주제에 대한 글, 이메일, 아이디어 등을 작성해 주세요.
3.  **계획 수립:** 특정 목표를 달성하기 위한 단계별 계획을 세워 주세요.
4.  **데이터 분석:** 제공된 데이터를 분석하거나 패턴을 찾아 주세요.
5.  **기타:** 원하시는 다른 종류의 요청을 해주세요.

원하시는 작업을 말씀해주시면 즉시 도와드리겠습니다.
