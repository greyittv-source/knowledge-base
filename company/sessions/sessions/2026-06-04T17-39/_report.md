# 📝 CEO 종합 보고서

📱 **가장 중요한 작업 우선순위 결정 보고**

현재 시스템은 사용자의 요청에 따라 가장 중요한 작업의 우선순위를 결정하고 실행 계획을 수립하는 데 집중합니다.

**분석 결과:**
제공된 정보는 사용자가 특정 작업을 요청하기 전의 **시스템 설정 및 지침**에 해당합니다. 따라서 현재 시점에서는 구체적인 실행할 작업이 제시되지 않았기 때문에, 가장 중요한 작업 우선순위를 결정할 수 없습니다.

**다음 단계:**
어떤 종류의 작업을 수행하기를 원하시는지 구체적인 지시(예: 콘텐츠 작성, 데이터 분석, 일정 관리, 특정 문서 요약 등)를 제공해 주시면, 해당 요청에 맞춰 가장 중요한 작업 우선순위를 결정하고 실행 계획을 수립해 드리겠습니다.

**요청 대기 중입니다.**
