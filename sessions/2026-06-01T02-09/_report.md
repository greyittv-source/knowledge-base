# 📝 CEO 종합 보고서

📱 **실행 결과**

현재 시스템은 사용자의 요청에 따라 내부적으로 정보를 분석하고, 설정된 지침에 따라 응답을 생성합니다.

---

**분석 결과:**

사용자님은 시스템의 **내부 상태(System State)**에 기반하여, 현재 시점에서 가장 중요한 다음 행동이나 정보 제공을 요청하고 계십니다.

**요청 사항:** 시스템이 현재 상태를 기반으로 다음 단계의 행동을 제안하거나, 내부 지침에 따라 정보를 요약/처리하는 것을 기대합니다.

**현재 상태 요약:**
현재 상태는 시스템이 제공받은 지침과 내부 데이터(System State)에 기반하고 있으며, 사용자가 직접적으로 처리해야 할 특정 작업 지시(예: 파일 생성, 외부 API 호출 등)는 포함되어 있지 않습니다.

**다음 단계 제안:**
현재 제공된 정보만으로는 구체적인 행동 지침이 없으므로, 시스템은 사용자가 다음으로 수행하고자 하는 목표를 명확히 제시해 주시기를 요청드립니다.

**다음 질문 예시:**
1. 현재 시스템 상태(System State)에 대해 요약해 드릴까요?
2. 특정 파일이나 정보에 대해 작업을 요청하시겠습니까?
3. 다음으로 수행하고 싶은 목표(Task)가 있으신가요?
