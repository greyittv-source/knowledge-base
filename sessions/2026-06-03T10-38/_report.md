# 📝 CEO 종합 보고서

📱 **분석 결과 및 실행 계획**

현재 시스템은 사용자 요청에 따라 **회사 내의 복잡한 역할과 컨텍스트**를 기반으로 최적의 실행 계획을 도출하도록 설계되어 있습니다.

**분석:**
사용자님은 시스템에 명시적으로 "회사 내의 역할"이나 "구체적인 작업 목록"을 제공하지 않았습니다. 대신, 시스템이 제공한 내부 지침(역할, 지침, 도구)을 바탕으로 **'자기 참조적 명령(Self-Referential Command)'**을 수행하도록 요청하셨습니다.

**실행 계획:**
제공된 지침에 따라, 현재 상황에서 가장 논리적이고 유용한 다음 단계는 **시스템의 현재 상태와 지침을 명확히 정리**하고, **사용자의 의도를 재확인**하는 것입니다.

**실행:**
현재 제공된 정보만으로는 구체적인 작업을 수행할 수 없으므로, 시스템의 역할에 기반하여 다음 단계를 제안합니다.

**다음 단계 제안:**

1. **역할 확인 요청:** 현재 시스템이 수행해야 할 구체적인 목표나 작업 목록을 제공해 주십시오. (예: "오늘 해야 할 업무 목록을 정리해 줘", "이 문서를 요약해 줘" 등)
2. **지침 확인 요청:** 시스템이 참고해야 할 핵심 지침(페르소나, 제약 조건, 목표)을 명확히 재확인해 주십시오.

이러한 정보를 제공해 주시면, 제공된 모든 컨텍스트(역할, 도구, 이전 대화 기록)를 활용하여 가장 정확하고 유용한 답변을 생성할 수 있습니다.
