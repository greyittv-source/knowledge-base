# 📱 영숙 — 회사 목표, 각 에이전트의 개인 목표, 최근 의사결정, 메모리를 검토하여 현재 가장 가치 있는 단일 작업 1개를 결정하고, 해당 작업에 대한 실행 계획을 수립하라.

📱 **분석 결과 및 실행 계획**

현재 제공된 정보만으로는 구체적인 실행 계획을 수립하기 어렵습니다. 요청하신 내용은 시스템 내의 복잡한 맥락(가장 최근의 작업 흐름, 사용자 목표 등)에 기반해야 정확한 답변이 가능합니다.

**요청하신 내용:** 현재 입력된 정보를 바탕으로 다음 단계에서 수행해야 할 구체적인 실행 계획을 제시해 달라는 요청으로 해석됩니다.

**현재 상황 진단:**
제공된 텍스트는 이전 대화의 맥락이나 특정 작업 지시가 아닌, 시스템 내부의 메타 정보(AI의 역할, 데이터 처리 방식 등)에 대한 설명으로 보입니다. 따라서, **실제 수행해야 할 구체적인 작업이나 목표가 명시되지 않았습니다.**

**다음 단계 제안:**
원하시는 구체적인 결과물을 얻기 위해서는 **수행하고자 하는 구체적인 목표나 질문**을 제시해 주시기 바랍니다.

예를 들어, 다음과 같은 질문을 해주시면 제가 요청하신 내용을 바탕으로 실행 계획을 수립할 수 있습니다.

1.  "오늘 내가 처리해야 할 가장 중요한 업무는 무엇인가요?"
2.  "최근에 논의했던 [특정 주제]에 대해 다음 단계는 무엇인가요?"
3.  "이 정보를 바탕으로 [특정 결과물]을 만들기 위한 단계별 계획을 짜주세요."

**👉 다음 질문을 기다리겠습니다.**
