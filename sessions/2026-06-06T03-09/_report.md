# 📝 CEO 종합 보고서

📱 **업무 분석 및 계획**

제공된 요청에 따라, 현재 시스템의 모든 정보를 분석하고 가장 우선순위가 높은 작업을 도출하여 실행 계획을 수립합니다.

**1. 목표 분석:**
사용자는 현재 시스템의 모든 정보를 종합하여 **가장 시급하고 중요한 다음 행동 계획**을 요청하고 있습니다. 이는 기존에 설정된 목표와 시스템의 맥락(특히 '업무 분석 및 계획'이라는 맥락)에 기반하여 다음 단계를 결정하는 것을 의미합니다.

**2. 시스템 상태 검토:**
현재 시스템은 사용자 입력(명령)을 기다리고 있으며, 이전 대화 기록이나 명시적인 목표 설정은 다음 요청을 위한 기반이 됩니다.

**3. 실행 계획:**
현재 명시적인 실행 요청이 없으므로, 시스템의 다음 단계를 제안하고 준비 상태를 알립니다.

**결론:**
명확한 다음 지시가 없으므로, 시스템은 다음 행동을 준비하고 사용자의 구체적인 요청을 기다립니다.

---
**다음 단계 제안:**

어떤 종류의 작업을 수행하기를 원하시나요? 예를 들어, 다음과 같은 요청을 해주실 수 있습니다.

1.  **기존 프로젝트 진행:** 이전에 논의했던 특정 프로젝트의 다음 단계(예: 콘텐츠 작성, 코드 리뷰, 전략 구상 등)를 요청합니다.
2.  **정보 검색/분석:** 특정 주제에 대한 정보 검색이나 데이터 분석을 요청합니다.
3.  **계획 수립:** 새로운 목표를 설정하고 그에 따른 상세한 실행 계획을 요청합니다.
4.  **시스템 점검:** 현재 시스템 설정이나 상태에 대한 점검을 요청합니다.

**구체적인 지시를 기다리겠습니다.**
